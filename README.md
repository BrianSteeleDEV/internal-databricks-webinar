# internal-databricks-webinar

## Overview
A collection of Apache Spark Structured Streaming examples written in Scala, created for a Databricks webinar. Covers basic streaming, static-stream joins, streaming-streaming joins, and the `foreachBatch` sink pattern.

## Technology Stack
- Scala: All streaming examples
- Apache Spark Structured Streaming: Core streaming engine
- Databricks: Target execution environment

## Notes
Internal Coretek educational material for a Databricks webinar. Demonstrates common Spark Structured Streaming patterns and join strategies.
